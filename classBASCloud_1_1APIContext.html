Detailed Description

BAScloud API context is a 1:1 implementation of the unabstracted API endpoints.

Methods in this class directly request the corresponding BAScloud API endpoint and return its unmodified response e.g. including error responses and raw JSON formatted response data.

It is recommended to use the abstracted class EntityContext which already handles error responses and JSON data parsing. For working with raw response data one may use this class directly.
