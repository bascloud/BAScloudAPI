Detailed Description

A Result structure representing API paging information.

Paging is structured so that there are two pointers, one for the next page when available, and one for the previous page when available, otherwise empty.

◆ count

size_t BAScloud::PagingResult::count = 1

The number of results remaining in the request result.

At currentPage==1, count describes the total number of entries of the result over all pages. When navigating to following pages, the count decreases to the remaining entries excluding the already seen.
